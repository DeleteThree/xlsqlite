ATTRIBUTION
================================================================================

xlsqlite is built upon infrastructure from the xlwings project:
https://github.com/xlwings/xlwings

The following components are derived from xlwings:
- COM server implementation (server.py)
- UDF registration and decorator system (udfs.py)
- Excel communication layer (main.py, _xlwindows.py, _xlmac.py)
- VBA wrapper generation
- Type conversion framework
- DLL interface files

Original xlwings license and copyright notices are preserved in the respective
source files.

The following components are original work developed for xlsqlite:
- SQL query parser (parser.py)
- Schema builder and type inference (schema.py)
- Query executor (executor.py)
- Error handling system (errors.py)
- Output formatter (output.py)
- Test suite
- Project-specific documentation
